# Administrator Hardening

De volledige administrator-hardening documentatie staat nu in:

- `HARDENING_DOCUMENTATION.md`

Daar vind je:

- wat er is gehard;
- hoe het technisch is geïmplementeerd;
- waarom elke maatregel nodig was;
- waarom dit de applicatie veiliger maakt;
- welke resterende risico's nog bestaan.

Dit bestand blijft alleen als korte verwijzing bestaan, zodat er één hoofdbron van waarheid is voor alle hardening-documentatie.
